# Zatürre(Pneumonia) Teşhis Destek Sistemi

Bu projeyi, göğüs röntgenlerini analiz ederek zatürre (pneumonia) belirtisi olup olmadığını tahmin etmek için hazırladım. 

### Nasıl Kullanılır?
* **Hızlıca denemek için:** [https://huggingface.co/spaces/boveric/zaturre-tespit-araci]
* **Bilgisayarda çalıştırmak için:** **Releases** bölümünden ZIP dosyasını indirin. 

### Teknik Detaylar
* **Model:** DenseNet121 mimarisi kullanıldı.
* **Dil:** Python 3.13 & TensorFlow.
* **Başarı Oranı:** %91 civarında doğru sonuç veriyor.

**Önemli Not:** Bu bir yardımcı araçtır, kesin teşhis için doktora danışılmalıdır.
